migrateDownloadedContent

public abstract Boolean migrateDownloadedContent(String contentName, String downloadedFolderName)

Migrate past downloaded content If the url value of contentData is a local address, contentName is empty, and downloadedFolderName is null, migration proceeds to the url value of contentData.

Return

true if the migration is successful, false otherwise.

Parameters

contentName

Content name which will be used for the name of downloaded content folder

downloadedFolderName

Content download folder name

Throws

PallyConException.ContentDataException

if contentData is wrong

PallyConException.MigrationException

if migration error

PallyConException.MigrationLocalPathException

When the localPath of ContentData and the path of the downloaded content are the same or a subdirectory
